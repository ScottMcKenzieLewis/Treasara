Treasara

Treasara is a bond valuation and analytics API designed to make fixed-income modeling approachable, transparent, and easy to experiment with. The project focuses on core fixed-income concepts such as yield, duration, convexity, and price sensitivity, providing a clean API that can be used for research, prototyping trading tools, or educational exploration of bond mathematics.

The goal of Treasara is not only to compute valuations, but to make them understandable. By pairing the API with a visual Angular interface, Treasara helps illustrate how bond prices respond to yield changes, coupon structures, and maturity profiles.

The project is open source and still evolving. You can explore the code, contribute ideas, or follow development on GitHub:
