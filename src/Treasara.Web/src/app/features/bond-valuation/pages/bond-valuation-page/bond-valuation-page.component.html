Non-Technical Description

This calculator estimates what a bond is worth today by looking at the payments it will make in the future. Bonds typically pay periodic interest (coupons) and return the principal at maturity. The calculator generates that schedule of future payments, ignores any payments that have already occurred, and then adjusts the remaining payments to reflect the time value of money.

Because money received in the future is worth less than money today, each future payment is discounted using the supplied interest rate (the curve rate). The discounted payments are then added together to produce the bond’s estimated present value.

Technical Description

Treasara values a fixed-rate bond by first generating the full coupon and principal cash-flow schedule based on the bond’s frequency, day-count convention, and roll convention. Cash flows occurring before the valuation date are excluded, leaving only the remaining payment schedule.

Each remaining cash flow is discounted back to the valuation date using a flat yield curve. The discount factor is calculated using the selected day-count convention and the supplied curve rate. The present values of all discounted cash flows are then summed to produce the bond’s total present value. The cash-flow breakdown table shows the individual payment date, cash-flow amount, discount factor, and present value for each remaining payment.
